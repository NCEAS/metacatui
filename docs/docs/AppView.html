showAlert_deprecated(msg, classesopt, containeropt, delayopt, optionsopt)

Previous to MetacatUI 2.14.0, the AppView#showAlert function allowed up to five parameters to customize the alert message. As of 2.14.0, the function has condensed these options into a single literal object. See the docs for AppView#showAlert. The old signature of five parameters may soon be deprecated completely, but is still supported.
