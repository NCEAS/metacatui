Class: CesiumVectorData

CesiumVectorData()

A CesiumVectorData Model is a vector layer (excluding Cesium3DTilesets) that can be used in Cesium maps. This model corresponds to "DataSource" models in Cesium. For example, this could represent vectors rendered from a Cesium GeoJSONDataSource. https://cesium.com/learn/cesiumjs/ref-doc/GeoJsonDataSource.html. Note: GeoJsonDataSource, CzmlDataSource, and CustomDataSource are supported. Eventually this model could support the KmlDataSource.
