addMdImage()

addMdImage - The function that gets called when a user clicks the custom add image button added to the markdown editor. It uses the UI created by the ImageUploaderView to allow a user to select & upload an image to the repository, and uses Woofmark's built-in add image functionality to insert the correct markdown into the textarea. This function must be called such that "this" is the markdownEditor view.

addTable(e, mode, chunks)

addTable - Creates the UI for editing and adding tables to the textarea. Detects whether the selection contained any part of a markdown table, then opens a woofmark dialog box and inserts a table editor view. If a table was selected, the table information is imported into the table editor where the user can edit it. If no table was selected, then it creates an empty table where the user can add data.
