Class: Cesium3DTileset

Cesium3DTileset()

A Cesium3DTileset Model is a special type of vector layer that can be used in Cesium maps, and that follows the 3d-tiles specification. See https://github.com/CesiumGS/3d-tiles

cesiumOptions

Options that are supported for creating 3D tilesets. The object will be passed to `Cesium.Cesium3DTileset(options)` as options, so the properties listed in the Cesium3DTileset documentation are also supported, see https://cesium.com/learn/cesiumjs/ref-doc/Cesium3DTileset.html
Type:
  • Object
Properties:
Name Type Description
ionAssetId string | number If this tileset is hosted by Cesium Ion, then Ion asset ID.
cesiumToken string If this tileset is hosted by Cesium Ion, then the token needed to access this resource. If one is not set, then the default set in the repository's AppConfig#cesiumToken will be used.
